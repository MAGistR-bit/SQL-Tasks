## 🎥 Как записать ASCIINEMA?
Если у вас еще нет записей (recording), то для начала 
установите [asciinema CLI](https://docs.asciinema.org/manual/cli/installation/).

Если asciinema установлена, то вы увидите соответствующую версию, выполнив команду ниже:
```bash
asciinema --version
```

Для записи терминала необходимо ввести команду:
```bash
asciinema rec demo.cast
```
Это запустит новый сеанс записи, во время которого все, 
что печатается на терминале, будет записано и сохранено в файле `demo.cast` 
в формате asciicast.

> Отмечу, что вместо `demo.cast` можно смело указывать свое название для файла.

Чтобы завершить сеанс записи, выйдите из консоли. 
Это можно сделать, нажав `ctrl+d` или введя команду `exit`.

Воспроизвести запись можно с помощью команды:
```bash
asciinema play demo.cast
```
Вы можете приостановить / возобновить воспроизведение, нажав пробел, 
или завершить воспроизведение досрочно, нажав `ctrl+c`.

Для того чтобы загрузить свою запись на `asciinema.org`, нужно указать:
```bash
asciinema upload demo.cast
```

Приведенная выше команда выведет секретную ссылку на страницу записи, 
где ее можно будет просмотреть.

Если вы уже загрузили запись, но не видите ее, 
то свяжите ее со своей учетной записью, запустив `asciinema auth` в своем терминале
на том же компьютере, который вы использовали для записи.

> Отмечу, что можно также проводить трансляции, используя asciinema.org
